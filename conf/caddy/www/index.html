Total Commander 群文件共享平台


由于 QQ 群文件下载不稳定，故搭建这个 FTP 平台，方便 Total Commander 群用户下载文件


访问此网站即可浏览、下载文件

https://ftp.totalcmd.co/


如果有特殊需要的话，可以使用 FTP 连接文件仓库

地址: ftp.totalcmd.co

// 文件管理员帐号
帐号: manager

// 博客管理员帐号
帐号: blogger

// 限制了同时连接的人数，如果你暂时连不上，可能其他人正在使用


上面的 guest 账户是只读账户，如果你需要上传文件的话，请在群内询问密码，或求助别人代为上传


本平台同时提供了 Web 上传、下载、管理功能，地址为 https://drive.totalcmd.co/，密码与 ftp 仓库一致


如果你已经持有密码，请不要公开，对于群内询问密码的，请自行甄别其社交信用，用私聊的方式交付密码

不过也不用太过保守，出问题及时联系我修改密码即可 (②群ID：黑帝)
